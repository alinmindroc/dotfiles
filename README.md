# Dotfiles

## Copy dotfiles
```
mkdir -p ~/.config/nvim
cp -r .config/nvim/init.vim ~/.config/nvim/

cp -r .zshrc ~

cp -r .bash_profile ~

cp -r .gitconfig ~
```

## Iterm2
1. Install iterm2
2. Preferences -> Profiles -> Keys -> Key Mappings -> Presets dropdown -> Import -> my.itermkeymap

## Zsh
1. Set zsh default shell
```
chsh -s /bin/zsh```
2. Install oh-my-zsh
